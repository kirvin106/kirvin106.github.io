Reporte de Instalación y Configuración de Apache Server

Introducción

Apache HTTP Server, comúnmente conocido como Apache, es un servidor web de código abierto desarrollado por la Apache Software Foundation. Es uno de los servidores web más populares y ampliamente utilizados en todo el mundo, sirviendo una gran parte de los sitios web activos en Internet.

La importancia de Apache en la web actual es inmensa. Proporciona una plataforma confiable y segura para alojar sitios web y aplicaciones web. Su arquitectura modular permite extender sus funcionalidades mediante módulos adicionales, lo que lo hace muy versátil para diferentes necesidades. Además, al ser de código abierto, es gratuito y cuenta con una amplia comunidad de desarrolladores que contribuyen a su mejora continua.

En este reporte, documentaremos los pasos seguidos para instalar y configurar Apache Server en un sistema operativo, incluyendo el proceso de cambiar el puerto predeterminado para adaptarlo a nuestras necesidades especificas.

Pasos de Instalación y Configuración

Paso 1: Verificar requisitos del sistema

Antes de instalar Apache, verificamos que nuestro sistema cumpla con los requisitos minimos. Esto incluye tener suficientes recursos de hardware y un sistema operativo compatible.

Paso 2: Descargar Apache Server

Accedimos al sitio web oficial de Apache (https://httpd.apache.org/) y descargamos la versión más reciente del servidor compatible con nuestro sistema operativo.

Paso 3: Ejecutar el instalador

descargamos en el sitio oficial de Apache, nos vamos al apartado de Windows y descargamos los binarios, le damos a "Apache Lounnge" y descargamos los binarios X64. Una vez descargados los extraemos y movemos la carpeta apache24 a nuestro disco local.

Paso 4: Configuración inicial

ya dentro de la carpeta de Apache buscamos donde nos diga bin y accedemos a ello.

Paso 5: buscamos httpd

comprobamos que tenemos httpd

# Buscamos la línea que contiene "Listen 80"
# y la cambiamos por el puerto deseado, por ejemplo:
Listen 8080

Localizamos el archivo httpd.conf en el directorio de instalación de Apache (generalmente en "conf/httpd.conf") y modificamos la línea correspondiente.

Paso 6: comprobamos la descarga y abrimos

una vez comprobamos, usamos el comando httpd-install, se abrira una ventana del firewall y le damos acceso. una vez instalado nos metemos a servicios y buscamos que este Apache24, iniciamos el servicio y comprobamos agregando localhost

Paso 7: cambio de puerto

hacemos el cambio de puerto desde la carpeta de Apache, buscamos la carpeta conf y buscamos httpd conf. cambiamos el lisen de 80 a 8080

Paso 8:revisamos

Finalmente, comprobamos que se haya echo el cambio y que se ingrese del link normal añadiendo lo del 8080.

Nota: Es importante recordar que al cambiar el puerto predeterminado, los usuarios deberán especificar el puerto en la URL para acceder al sitio (ej: http://misitio.com:8080). Además, debemos asegurarnos de que el firewall permita el trafico en el nuevo puerto seleccionado.

Conclusión

La instalación y configuración de Apache Server es un proceso relativamente sencillo que nos permite tener un servidor web funcional en poco tiempo. El cambio de puerto es una configuración útil cuando necesitamos ejecutar múltiples servidores web en la misma máquina o cuando el puerto 80 está siendo utilizado por otra aplicación.

Apache continúa siendo una opción popular para alojar sitios web debido a su estabilidad, flexibilidad y el fuerte soporte de la comunidad. Con esta instalación, ahora tenemos una base sólida para desarrollar y probar aplicaciones web.
